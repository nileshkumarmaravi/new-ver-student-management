#

The **Student Management System** is a command-line based project developed using **Bash scripting in Linux**. It allows users to manage student records from the terminal through a menu-driven interface.

The system can add, search, view, and remove student records stored in a text file. It also includes features like **automatic roll number generation, student dashboard, backup system, CSV export, input validation, and colored output** for a better command-line experience.

This project helps in learning **Linux, Bash scripting, file handling, and basic CLI application development**.
